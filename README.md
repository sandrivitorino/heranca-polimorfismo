# Module: Inheritance and Polymorphism

In this module I studied important Object-Oriented Programming concepts in Java, focusing on **inheritance, polymorphism and abstract classes**.

These concepts are fundamental to build scalable and well-structured systems using object-oriented design.

## Topics studied

- Inheritance
- Code reuse through superclass and subclass
- Method overriding
- Polymorphism
- Upcasting and downcasting
- Abstract classes
- Abstract methods
- Designing flexible object-oriented systems

## What I learned

During this module I practiced how classes can **inherit behavior from other classes**, allowing code reuse and better organization.

I also learned how **polymorphism** allows different objects to respond to the same method call in different ways depending on the object type.

Another important concept studied was **abstract classes**, which allow defining a base structure that other classes must follow.

## Exercises

To reinforce the concepts, I implemented exercises involving:

- Creating superclass and subclasses
- Implementing method overriding
- Using polymorphism with collections of objects
- Implementing abstract classes and abstract methods

These exercises helped me better understand how these concepts work in real Java applications.
